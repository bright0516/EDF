# plotEDF是画地震震中分布图模块，目前模块共有3个功能：
  1.画单个时间段地震分布（地震根据时间早晚用不同颜色标注）；
  2.画两个时间段地震分布（第一段时间地震用红色标记，第二段时间地震用蓝色标记）；
  3.统计单个时间段内地震分震级档地震次数；
  
# 参数解释
  figType #图件类型，1：单个时间段地震分布；2：两个时间段地震分布               
  startDate #目录开始时间，单个时间段，仅单个时间段需要
  endDate #目录结束时间，单个时间段，仅单个时间段需要
  startDate1 #第一段目录开始时间，两个时间段，仅两个时间段需要
  endDate1 #第一段目录结束时间，两个时间段，仅两个时间段需要
  startDate2 #第二段目录开始时间，两个时间段，仅两个时间段需要
  endDate2 #第二段目录结束时间，两个时间段，仅两个时间段需要
  maxLat #最大纬度
  minLat #最小纬度
  maxLon #最大经度
  minLon #最小经度
  minMag #最大震级
  maxMag #最小震级
  figName #图件名称
